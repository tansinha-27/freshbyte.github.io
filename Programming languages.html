COMMON PROGRAMMING LANGUAGES



C/C++

C and C++ are very similar. C++ is an extension of C , also known as 'C with classes'. Developed by Bjarne Strousroup, C++ is a fast programming language, supporting Object Oriented Programming(OOP).Used in competitive programming, game development,operating systems. It is a generic purpose programming language with high level features , and can be compiled on almost every platform.It is a very popular choice as it delves deep into the core basics of programming.




JAVA

Java is used in creating mobile apps, web apps and is one of the most used languages in today's times.It is a high level, object oriented programming language and the compiled code runs on almost all operating systems.It was developed by Sun Microsystems and James Oak is known as the father of Java.The syntax of Java is similar to C/C++, and is a must for application developers.




PYTHON

The most popular choice for beginners and data enthusiasts, Python is a language which has a simple syntax, but the scope is immense.Used in data science, machine learning,interaction with databases,GUI programming, it has broad libraries and immense versitality. It is very similar to normal English, and supports structural and object oriented programming.Developed by Guido Van Rossum ,it's easy to learn and read and it's source code is available to general public!




JAVASCRIPT

The "language of the web",Javascript is used to make websites interactive.One of the most popular languages,and is a full-fledged dynamic programming language.It supports a large amount of frameworks and libraries, which make websites easy to use and interactive.Even API's can be incorporated to implement functionality using JS.Along with HTML and CSS, it's possibilities become endless.It is high level and event driven.




C#

C sharp is mainly used for game and app development.Developed by Microsoft, it is object oriented , functional and general purpose language.It is simple,modern and portable, and runs on .Net framework.Applications are vastly developed using C#.




PHP

PHP Hypertext Processor, abbreviated as PHP,is used majorly in web applications to interact with a variety of databases such as MySQL, Oracle, Sybase.It is a backend scripted language usually integrated with HTML. With a similar syntax to C, it's easy to use, simple and efficient. You can do a lot with PHP especially while making web apps, such as huge scale e-commerce websites. It can manage and set cookies, encrypt data ,modify and delete data as per the users needs.Highly recommended for web development




GO

Developed and open sourced by three programmers at Google, Go/Golang is a relatively newer programming language.A procedural programming language,it aims to be modern and easy, attemoting to use typing and cluttering.It gives a lot of control over memory, one of the key features which makes it apart from other languages.It has made it's way to being one of the most important programming languages in the future.
