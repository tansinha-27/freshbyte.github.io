TERMS YOU MIGHT ENCOUNTER




Algorithm

Algorithm is a set of structural steps used to solve a particular problem.Eg:Search Sort

API

Application Programming Interface is a software which allows data transfer from one software to another while following certain protocols.Eg:Gmail SignIn API

Application Server

Application servers act as a connect between web servers(which directly communicate with clients)and database servers(which store information). Eg:Apache

Backend

Backend refers to the working or behind the scenes of a website or software, where all the interaction and processing takes place.It is not visible to the user.Eg:REST API

Big Data

Big Data refers to data that is so large and complex that it cannot be processed or analysed using traditional methods.

Bit/Byte

A bit refers to either 0 or 1.Different combinations of bits help computers to store large amount of information. One byte is equal to 8 bits.

Bug

Bugs are errors or faults which one can encounter while using/developing a software or program, which tamper with their functionality.

Cloud Computing

Cloud computing refers to renting of various services such as storage, databases,networking, collaboration etc.over the Internet, delivered via a platform.Eg:Microsoft Azure

DBMS

DataBase Management System is used to store and manage tons of data in a systematic manner, which can be retrieved and manipulated. Eg:MySQL

Data structures

These are used to organize and store data in a manner which can be used and accessed efficiently.Eg:Array

Framework

These allow development of applications in an easier and more efficient manner, taking care of low-level functionality themselves.Developers can then focus on the more complex and actual functionality of the program.Eg:Django

Frontend

This is the visible part of a website/application, something which the user can see and interact with.

Full Stack

A combination of technologies to make both server side and client side functional i.e. frontend and backend. Eg:MERN stack

Internet

The Internet is a globally connected system offering a variety of services through connected networks , which allow transmission of information worldwide.

IP Address

It is a unique address assigned to each and every website , which helps the server to find it whenever a user tries to access it.IP stands for Internet Protocol.

Library

This is a collection of predefined class templates and functions to promote code reuse, so that the developer does not have to write the code again and again.Eg:TensorFlow

Machine Language

It is a language comprising of bits which the computer can read and understand, as the computer cannot understand high level programming languages.

OOP

Object Oriented Programming is a model of programming which has objects and classes. It is efficient,clean, helps in code reusability and fast.Eg:C++

Open Source

Open Source (OSS) is code which is available and accessible to general public,who can view,make valuable changes and distribute it.

Operating System

O/S is a software consisting of a collection of programs which help in file management, memory allocation, control peripheral devices and help to run the computer system . Eg: Linux

UI

User Interface(UI) defines the aesthetic elements which a user sees so that the frontend seems appealing to the user.

UX

User Experience (UX) is the interaction a user has with the product / service. Good UX results in a better service and better customer network, which in turn helps to grow the business.

Web Browser

It is a software which accesses ,finds and displays web content from anywhere on the Internet.Eg:Google Chrome
